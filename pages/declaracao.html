Esta é uma declaração de acessibilidade do Foco Acessível.

Medidas para apoiar a acessibilidade

O Foco Acessível toma as seguintes medidas para garantir a acessibilidade do Foco Acessível:

  • Incluir a acessibilidade como parte de nossa declaração de missão.
  • Incluir acessibilidade em todas as nossas políticas internas.
  • Fornecer treinamento contínuo de acessibilidade para nossa equipe.
  • Atribuir metas e responsabilidades claras de acessibilidade.
  • Empregar métodos formais de garantia de qualidade de acessibilidade.

Status de conformidade

As Diretrizes de Acessibilidade para Conteúdo Web (WCAG) definem requisitos para designers e desenvolvedores a fim de melhorar a acessibilidade para pessoas com deficiência. Elas definem três níveis de conformidade: Nível A, Nível AA e Nível AAA. A Foco Acessível está totalmente em conformidade com a WCAG 2.1 Nível AA. "Totalmente em conformidade" significa que o conteúdo está totalmente de acordo com o padrão de acessibilidade, sem exceções.

Especificações técnicas

A acessibilidade do Foco Acessível depende das seguintes tecnologias para funcionar em combinação com o navegador da web e quaisquer tecnologias assistivas ou plugins instalados no seu computador:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Essas tecnologias são utilizadas para garantir conformidade com os padrões de acessibilidade adotados.
